## 恢复方式

如需回滚 iOS 部分:

```bash
cp -R _backup_旧版本_2026-05-13_liquidglass前/ios/TuTuVisionApp/* ios/TuTuVisionApp/
```

如需重新安装后端依赖:

```bash
cd _backup_旧版本_2026-05-13_liquidglass前/backend
npm install
```
